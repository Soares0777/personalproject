# Projeto CRUD com Upload de Arquivos

Sistema CRUD completo com integração a banco de dados e upload de arquivos, desenvolvido para fins de estudo e prática.

## Funcionalidades

 Criar, listar, editar e excluir registros
 Upload e armazenamento de arquivos
 Integração com banco de dados
 Validação de dados

## Tecnologias Utilizadas

Backend: PHP
Frontend: HTML, CSS
Banco de Dados: MySQL
Upload de arquivos

## Objetivo
Projeto desenvolvido para aprimorar conhecimentos em CRUD, banco de dados e manipulação de arquivos no backend.
